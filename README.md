# Cartita

Pequeño proyecto interactivo que muestra un sobre romántico con carta animada, música de fondo y efectos visuales. El usuario puede abrir el sobre haciendo clic y también cerrarlo al hacer clic fuera del sobre.

## Cómo usar

1. Abrir `index.html` en un navegador moderno.
2. Hacer clic en el sobre para abrirlo.
3. Hacer clic fuera del sobre para cerrarlo.

## Qué debe explicar este README después de cada commit

Después de cada commit, documenta aquí brevemente qué se hizo y por qué. Esto ayuda a mantener un historial claro y a entender la evolución del proyecto.

### Formato recomendado para cada commit

- `feat:` cuando se agrega una funcionalidad nueva.
- `fix:` cuando se corrige un error.
- `style:` cuando solo se cambian estilos sin lógica.
- `docs:` cuando se actualiza la documentación.
- `refactor:` cuando se reorganiza código sin cambiar comportamiento.

### Ejemplo de nota de commit

- `feat: cerrar el sobre al hacer clic fuera del contenedor`
- `fix: evitar que el texto de la carta se sobreponga al cerrar el sobre`
